Referências HTML

  • <applet/>

    Exibe um applet Java dentro da página HTML.

  • <area/>

    Define uma área clicável dentro de um mapa de imagem.

  • <aside/>

    Define um conteúdo que é relacionado ao conteúdo em volta, mas não é parte dele.

  • <br/>

    Insere uma quebra de linha.

  • <canvas/>

    Usada para desenhar gráficos, em tempo real, através de scripts (geralmente JavaScript).

  • <caption/>

    Define uma legenda de tabela.

  • <caption/>

    Define uma legenda de tabela.

  • <cite/>

    Define o título de um trabalho.

  • <dd/>

    Define uma descrição de um item em uma lista de definição.

  • <del/>

    Define um texto que foi deletado de um documento.

  • <div/>

    Define uma divisão ou uma seção em um documento HTML.

  • <form/>

    Define um formulário HTML para entrada de dados do usuário.

  • <hr/>

    Define uma quebra temática no conteúdo.

  • <ins/>

    Define um texto que foi inserido em um documento.

  • <kbd/>

    Define um texto que deve ser inserido pelo usuário.

  • <main/>

    Especifica o conteúdo principal do documento.

  • <nav/>

    Define um conjunto de links de navegação.

  • <ol/>

    Define uma lista ordenada.

  • <output/>

    Define o resultado de um cálculo.

  • <select/>

    Cria uma lista suspensa.

  • <q/>

    Define uma citação curta.

  • <sup/>

    Define o texto sobrescrito.

  • <tfoot/>

    É usada para agrupar o conteúdo do rodapé em uma tabela HTML.

  • <ul/>

    Define uma lista não ordenada.

  • <wbr/>

    Define uma possível quebra de linha.
