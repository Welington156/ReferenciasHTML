Referências HTML

  • <main/>

    Especifica o conteúdo principal do documento.

  • <caption/>

    Define uma legenda de tabela.

  • <sup/>

    Define o texto sobrescrito.

  • <select/>

    Cria uma lista suspensa.

  • <div/>

    Define uma divisão ou uma seção em um documento HTML.

  • <tfoot/>

    É usada para agrupar o conteúdo do rodapé em uma tabela HTML.
