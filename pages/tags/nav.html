Tag <nav/>

Definição

A <nav> tag define um conjunto de links de navegação.

Use esta tag junto com a tag <a> para definir os links de navegação.

Navegadores, como leitores de tela para usuários com deficiência, podem usar esse elemento para determinar se devem omitir a renderização inicial desse conteúdo.

Atributos

A <nav> tag também suporta atributos globais em HTML

Exemplos

            
<nav>
    <a href="https://www.google.com">Google</a>
    <a href="https://www.youtube.com">Youtube</a>
</nav>
