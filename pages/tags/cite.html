Tag <cite/>

Definição

A <cite> tag define o título de um trabalho criativo (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.).

O nome de uma pessoa não é o título de uma obra.

O texto no elemento <cite> geralmente é renderizado em itálico.
