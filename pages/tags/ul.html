Tag <ul/>

Definição

A <ul> tag define uma lista não ordenada (com marcadores).

Use a <ul> tag junto com a <li> tag para criar listas não ordenadas.

Use CSS para estilizar listas.

Para listas ordenadas, use a <ol> tag.
