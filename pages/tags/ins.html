Tag <Ins/>

Definição

A <ins> tag define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido.

Atributos

Atributo Valor Descrição
cite URL Especifica uma URL para um documento que explica o motivo pelo qual o texto foi inserido/alterado
datetime YYYY-MM-DDThh:mm:ssTZD Especifica a data e hora em que o texto foi inserido/alterado
