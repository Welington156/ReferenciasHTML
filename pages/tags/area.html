Tag <area/>

Definição

A <area>tag define uma área dentro de um mapa de imagem (um mapa de imagem é uma imagem com áreas clicáveis).

<area>os elementos são sempre aninhados dentro de uma <map>tag.

O atributo usemap em <img>está associado ao atributo <map>do elemento name e cria uma relação entre a imagem e o mapa.

Atributos

Atributo Valor Descrição
alt text Especifica um texto alternativo para a área. Obrigatório se o atributo href estiver presente
coords coordinates Especifica as coordenadas da área
download filename Especifica que o destino será baixado quando um usuário clicar no hiperlink

Exemplos

            
                <img src="workplace.jpg" alt="Workplace" usemap="#workmap" width="400" height="379">
                <map name="workmap">
                <area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm">
                <area shape="rect" coords="290,172,333,250" alt="Phone" href="phone.htm">
                <area shape="circle" coords="337,300,44" alt="Cup of coffee" href="coffee.htm">
                </map>
