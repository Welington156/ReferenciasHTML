Tag <ol/>

Definição

A <ol> tag define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

A <li> é usada para definir cada item da lista.

Atributos

Atributo Valor Descrição
reversed reversed Especifica que a ordem da lista deve ser invertida (9,8,7...)
start number Especifica o valor inicial de uma lista ordenada
type 1
a
A
i
I		 Especifica o tipo de lista ordenada

Exemplos

            
<ol>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ol>
