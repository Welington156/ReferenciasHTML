Tag <canvas/>

Definição

A <canvas> é usado para desenhar gráficos dinamicamente, via JavaScript.

A <canvas> elemento é apenas um contêiner para gráficos. Você deve usar JavaScript para realmente desenhar os gráficos.

Canvas tem várias aplicações, como desenho de gráficos, renderização de gráficos, processamento de fotos ou criação de animações.

Atributos

Essa tag suporta todos os atributos globais

Exemplos

                
<canvas> id="myCanvas" width="200" height="100" style="border:1px solid #000000;"></canvas>
