Tag <output/>

Definição

A <output> tag representa o resultado de um cálculo ou o resultado de uma ação do usuário.

Atributos

Atributo Valor Descrição
for id Especifica a relação entre o resultado e o elemento que o gerou
form form_id Especifica um ou mais formulários que o elemento faz parte
name name Especifica o nome do elemento

Exemplos

            
<form oninput="x.value=parseInt(a.value)+parseInt(b.value)">
