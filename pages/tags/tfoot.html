Tag <Tfoot/>

Definição

A <tfoot> tag é usada para agrupar o conteúdo do rodapé em uma tabela HTML.

O <tfoot> elemento é usado em conjunto com os elementos <thead> e <tbody> para especificar cada parte de uma tabela (rodapé, cabeçalho, corpo).

Por padrão, uma legenda de tabela será alinhada ao centro acima de uma tabela. No entanto, as propriedades CSS text-align e caption-side podem ser usadas para alinhar e colocar a legenda.

Atributos

Atributo Valor Descrição
Não contem Não contem Não contem

Exemplos

            
                <table>
                <thead>
                  <tr>
                    <th>Month</th>
                    <th>Savings</th>
                  </tr>
                </thead>
                <tbody>
                  <tr>
                    <td>January</td>
                    <td>$100</td>
                  </tr>
                  <tr>
                    <td>February</td>
                    <td>$80</td>
                  </tr>
                </tbody>
                <tfoot>
                  <tr>
                    <td>Sum</td>
                    <td>$180</td>
                  </tr>
                </tfoot>
              </table>
