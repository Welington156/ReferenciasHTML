Tag <kbd/>

Definição

A <kbd>tag é usada para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Esta tag não está obsoleta. No entanto, é possível obter um efeito mais rico usando CSS (veja o exemplo).

Atributos

Atributo Valor Descrição
Não contem Não contem Não contem

Exemplos

            
                kbd {
                    border-radius: 2px;
                    padding: 2px;
                    border: 1px solid black;
                  }
                <p>Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to copy text (Windows).</p>
                <p>Press <kbd>Cmd</kbd> + <kbd>C</kbd> to copy text (Mac OS).</p>
